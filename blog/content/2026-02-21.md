I learned a lesson: don't try to save a buck by buying cheap AliExpress no-name garbage. I bought some knock-off BME280 sensor which fried about 24 hours after I installed it. All that soldering for nothing! (Sad thing is, this isn't the first time it happened to me)

## New sensor
I had an M5Stack ENV III module with a SHT30 and QMP6988 sensor in it (from a previous project), and I used it to replace the dead BME280. It's been working beautifully for a few days now and no problems at all. I should have used this in the first place instead of waiting 2 weeks for a crummy sensor to arrive from China, only for it to die 24 hours later. Lesson learned. 

## Smoothed AQI data
The AQI was updating every few seconds, causing inconsistent readings. Every time the wind blew from a different direction, (local fires, motorbike exhaust, etc), the AQI could jump from 60 to 130, back down to 90, and so on. This wasn't a very accurate reading, so I smoothed out the data. Now, every 60 seconds, it takes an average of the last 20 samples. The website will update every 60 seconds with a new reading, showing the average of the last 20 minutes.
